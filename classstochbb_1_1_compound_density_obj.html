Detailed Description

Implements the generic density of all compound random variables.

That is, a random variable that is distributed according to some parametric density where the parameters are random variables too.

The compound distribution is obtained by numerical integration over the complete parameter space. The bounds for this integration are obtained for each parameter using the stochbb::Density::rangeEst method for each parameter density. For the sake of efficientcy, delta-distributed parameters are handled separately, reducing the dimension of the parameter space to intergrate over.

double &  a,
double &  b 
) const
virtual

Estimates the range on which the PDF and CDFs need to be evaluated based on the -quantile of atomic distributions.

Please note that this value is not an estimate of -quantiles of the distribution. It reflects the interval on which all distributions, this distribution depends on, need to be evaluated for the evaluation of this distribution.

Parameters
alphaSpecifies the quantiles used for the estimation of the evaluation range.
aThe lower bound estimate for the evaluation range.
bThe upper bound estimate for the evaluation range.
