Detailed Description

Here you find all classes and functions that are part of the public API.

StochBB implements a simple mark & sweep garbage collector (GC, Memory Management) for freeing all unreachable objects. For this GC to work, all directly reachable objects are held within containers (classes derived from stochbb::Container). These container classes form the API of StochBB and are listed below.
