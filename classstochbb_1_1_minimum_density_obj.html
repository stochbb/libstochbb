void MinimumDensityObj::rangeEst ( double  alpha,
double &  a,
double &  b 
) const
virtual

Estimates the range on which the PDF and CDFs need to be evaluated based on the -quantile of atomic distributions.

Please note that this value is not an estimate of -quantiles of the distribution. It reflects the interval on which all distributions, this distribution depends on, need to be evaluated for the evaluation of this distribution.

Parameters
alphaSpecifies the quantiles used for the estimation of the evaluation range.
aThe lower bound estimate for the evaluation range.
bThe upper bound estimate for the evaluation range.
