Command line interface

The StochBB installation comes with a command line tool.

With this tool, a process definition in XML can be analyzed. The results are returned as CSV or they can be plotted directly.

Synopsis

stochbb [OPTIONS] [CMD] INPUTFILE [OUTPUT OPTIONS]

Options

  • --help Prints a short help string and exits.
  • --version Prints the version string and exits.
  • --log-debug Prints debug messages to stderr. By default, only warnings and error messages are printed to stderr.

Commands

  • pdf Specifies to evaluate the PDF of the variables selected in the INPUTFILE (default).
  • cdf Specifies to evaluate the CDF instead of the PDF of the variables selected in the INPUTFILE.
  • sample Samples from the variables selected in the INPUTFILE.

Output options

  • --plot Plots the PDF or CDF of the output variables specified in the file INPUTFILE.
  • --csv=FILENAME Writes the PDF or CDF of the output variables specified in the INPUTFILE to FILENAME.

Examples

For example, to plot the PDF of the selected random variables specified in the "example.xml" process description file

stochbb pdf example.xml --plot

The same PDF can be stored as CSV in "output.csv" by calling

sotchbb pdf example.xml --csv=output.csv

The call

sotchbb pdf example.xml

will print the CSV data to stdout.
