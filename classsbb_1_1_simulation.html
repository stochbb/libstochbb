Detailed Description

Collects several variable definitions and which PDFs are evaluated.

The simplest way to construct a Simulation is to parse a simulation specification from xml e.g.

Simulation sim = Simulation::fromXml("filename.xml");

The matrix gets resized. Each column represents a output variable where the first column is time. Each row represetns a time-point at which the CDFs are evaluated.
