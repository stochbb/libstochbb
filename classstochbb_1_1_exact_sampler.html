Detailed Description

Implements a sampler for several possibly dependent random variables.

This class should be used if one is interesed in sampling several random variables simultaneously. If only one variable of a very large system of random variables is sampled, consider also the MarginalSampler class.

void ExactSampler::sample ( Eigen::Ref< Eigen::MatrixXd >  out) const

Samples from the random variables passed to the constructor.

The number of row of out specify the number of samples to draw. The number of columns must match the number of random variables passed to the constructor, i.e. each colum corresponds to a random variable passed to the constructor.
